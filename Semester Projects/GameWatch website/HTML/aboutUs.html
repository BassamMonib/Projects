ABOUT GAME WATCH

This website will provide information about latest game releases and tell about their status (e.g. Released/Unreleased, Cracked/Uncracked, AAA/Indie etc.). It will also tell about which sale is currently going on for which gaming platform (Steam, EA, Epic etc.). It will also allow user to make his account (sign-up) and then log-in into his account so that the user can be notified about the above told information. User can see all games currently available on our website and can also search for its required game using our website search module. Our website will also allow the user to buy games and also use some discount coupons (only valid ones). User can also add cards for purchasing games. But this will only be available to those users who have logged into our website.
